What problem does the following code solve?

var myString = 'This sentence has has a fault, and and we need to fix it.';
var myRegExp = /(\b\w+\b) \1/g;
myString = myString.replace(myRegExp, '$1');

The above code matches and replaces the repeated words, "has has" and "and and."

Now, imagine that you have the following:
var myRegExp = new RegExp('(\b\w+\b) \1');
Why would this not work, and how could you fix it?

This would not work because a constructor object has been created with a string parameter. To fix this, add two backslashes, not one, before special characters.
Moreover, the global match ('g' modifier) should be passed to the second parameter, that the program does not stop after the first matched result.
